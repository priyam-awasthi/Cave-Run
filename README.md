# Cave-Run
Cave Run is a 2-player cooperative puzzle-platformer where players solve puzzles and collect gems to escape a mysterious cave. 

Players:
- Bob: Rookie adventurer
- Joe: Experienced guide

Controls:
- Bob: W - Jump, A - Move Left, D - Move Right
- Joe: Up Arrow - Jump, Left Arrow - Move Left, Right Arrow - Move Right

Objective: Collect all gems and reach the exit together.
Story: Bob and Joe explore a cave near their home, get trapped, and must work together to escape.
